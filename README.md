## Installation and Setup

### Hardware Setup
- Connect bedside units (buttons and indicators) to the microcontroller  
- Connect communication modules (RS485 / Ethernet) between nodes  
- Set up nurse station display (monitor or LCD)  
- Connect power supply to all modules  
- Ensure proper wiring for long-distance communication  

### Firmware Setup
- Open the project in Arduino IDE or relevant IDE  
- Select the appropriate microcontroller board (e.g., Arduino Mega, ATmega328P)  
- Configure communication protocol (Modbus / Serial / Ethernet)  
- Upload firmware to all slave and master devices  

### System Setup
- Assign unique room IDs to each bedside unit  
- Configure master device to receive and display messages  
- Test communication between all nodes  
- Verify button press events and indicator responses
